Architecture FTTH — Fibre Passive (PON)

Le FTTH (Fiber To The Home) est l'architecture fibre optique qui tire un câble de verre jusqu'au domicile ou au bureau de l'abonné. C'est une architecture passive : aucun équipement électronique actif n'est nécessaire entre le NRO (Nœud de Raccordement Optique) et l'abonné. Le signal lumineux se propage directement.

La technologie GPON (Gigabit PON) permet à un seul émetteur OLT de desservir jusqu'à 64 abonnés via un coupleur optique passif. La prochaine génération XGS-PON monte jusqu'à 10 Gbps symétrique.

  • Architecture passive : pas de panne électronique sur le réseau de distribution
  • Débits symétriques : jusqu'à 10 Gbps descendant ET montant
  • Faible atténuation : signal stable sur de longues distances (jusqu'à 20 km)
  • Mutualisée : jusqu'à 64 abonnés par fibre OLT (splitter passif)
  • Latence ultra-faible : < 5ms pour les applications temps réel

Glossaire des équipements FTTH

OLT — Optical Line Terminal

  • Équipement actif situé au NRO
  • Émet le signal laser vers les abonnés
  • Gère la synchronisation TDMA des ONTs
  • Connexion réseau cœur Orange (10G/100G)
  • 1 port OLT → jusqu'à 64 abonnés (GPON)

Splitter — Coupleur passif

  • Divise le signal optique sans électricité
  • Ratios : 1:2, 1:4, 1:8, 1:16, 1:32, 1:64
  • Atténuation : −3 dB par doublement
  • 1:32 = −15 dB d'atténuation
  • Installé en PMZ extérieur ou PBO palier

PBO — Point de Branchement Optique

  • Boîtier d'immeuble en pied de colonne
  • Contient le coupleur 1:16 ou 1:32
  • Permet le branchement des câbles drop
  • Accès technicien Orange uniquement
  • Connexion vers chaque palier via jarretière

ONT — Optical Network Terminal

  • Convertit signal optique ↔ signal électrique
  • Installé chez l'abonné (proche PTO)
  • Ports LAN RJ45 GigabitEthernet
  • LED : PWR / PON / LOS / LAN
  • Configuré et authentifié par OLT via PLOAM

PTO — Prise Terminale Optique

  • Prise fibre murale chez l'abonné
  • Connecteur SC/APC (vert) standard Orange
  • Terminaison du câble drop G.657
  • Point de démarcation réseau Orange/client
  • Nettoyage obligatoire avant connexion

OTDR — Réflectomètre optique

  • Mesure l'atténuation sur la fibre
  • Détecte les cassures et mauvaises jonctions
  • Identifie la distance des défauts
  • Budget max GPON : −28 dB total
  • Résultat archivé dans dossier technique
Notice d'installation FTTH — Étape par étape
01

Vérification du tirage existant et du PBO

Identifier le PBO (Point de Branchement Optique) de l'immeuble, vérifier qu'une fibre est bien disponible pour le logement/bureau concerné.

Vérifications terrain PBO
✓ Localiser le PBO (cave, local technique, pied de colonne) ✓ Ouvrir le PBO avec la clé triangulaire Orange ✓ Identifier la fibre disponible selon le bon de commande ✓ Vérifier l'état du coupleur (pas de coupure, propre) ✓ Repérer le numéro de port dédié au logement/bureau ✓ Documenter avec photos (numéro PBO, port utilisé)
02

Passage du câble drop G.657A2

Tirer le câble de branchement (câble drop) depuis le PBO jusqu'à la PTO dans le logement. Le câble G.657A2 est conçu pour les courbures serrées en immeuble.

Passage câble drop — Règles de tirage
CÂBLE : G.657A2 — Rayon de courbure min. 7,5 mm GAINE : Passage dans goulotte ou sous moulure PVC ✓ Tirer doucement (tension max 80N) ✓ Ne pas croiser les câbles, lovage propre ✓ Respecter les courbures : angle 90° avec coude PVC ✓ Fixer tous les 50 cm avec collier de câblage ✓ Étiqueter les deux extrémités : N° logement / date ⚠ Ne jamais écraser ni plier à moins de 7,5 mm de rayon ⚠ Tester la continuité optique avant de fermer les caches
03

Installation de la PTO (Prise Terminale Optique)

Fixer la boîte de PTO sur le mur, y épissurer ou connecter le câble drop, et y monter le connecteur SC/APC.

Montage PTO — Procédure
✓ Fixer la boîte PTO sur le mur (hauteur 30 cm du sol min.) ✓ Dénuder le câble drop sur 10 cm ✓ Nettoyer la fibre nue à l'alcool isopropylique 99% ✓ Cliver proprement avec le cleaver (angle <0.5°) ✓ Insérer dans connecteur SC/APC pré-poli (vert) ✓ Mesurer la réflectance : < −60 dB (SC/APC) ✓ Langer et lover l'excédent de fibre dans la PTO CONNECTEUR : SC/APC (biseau 8°) — Couleur VERTE ✓ Nettoyer chaque connecteur avant toute connexion ✓ Utiliser stylet de nettoyage One-Click Cleaner
04

Connexion au PBO et jarretière optique

Brancher la jarretière optique depuis le port PBO vers le câble drop, en respectant le bon port selon le plan de câblage.

Raccordement PBO
✓ Identifier le port PBO alloué (bon de commande) ✓ Nettoyer le connecteur de la jarretière (One-Click) ✓ Vérifier le connecteur côté câble drop (SC/APC propre) ✓ Connecter jarretière → port PBO dédié ✓ Connecter côté câble drop → port sortie PBO logement ✓ Refermer le PBO proprement (pas de câble coincé) ℹ Jarretière standard Orange : SC/APC — SC/APC — 1m ℹ Couleur jaune (OS2 monomode)
05

Mesures OTDR et powermètre

Effectuer les mesures obligatoires pour valider la qualité de la liaison optique avant d'installer l'ONT.

Mesures optiques — Seuils GPON Orange
POWERMÈTRE (depuis PTO, source OLT Orange) : Puissance reçue : entre −8 dBm et −27 dBm En dehors → vérifier jonctions, splitter, câble OTDR (mesure trajet complet) : Atténuation totale : < 28 dB (budget GPON) Perte par jonction : < 0,5 dB Perte par connecteur : < 0,5 dB Réflectance connecteur : < −50 dB RÉSULTATS À ARCHIVER : ✓ Capture écran OTDR (fichier .SOR) ✓ Valeur powermètre notée dans CR ✓ Photos de chaque jonction ⚠ Si puissance < −27 dBm → ONT ne synchronisera pas
06

Installation et configuration de l'ONT

Brancher l'ONT (Optical Network Terminal) sur la PTO, le connecter au réseau local du client et vérifier sa synchronisation avec l'OLT.

Installation ONT — Séquence de démarrage
1. Brancher câble SC/APC PTO → port PON de l'ONT (couleur bleue) 2. Alimenter l'ONT (cordon secteur) 3. Attendre la séquence de boot (~60 secondes) SÉQUENCE LED ATTENDUE : PWR → 🟢 Vert fixe (alimentation OK) PON → 🟡 Clignotant → 🟢 Vert fixe (sync GPON OK) LOS → 🔴 Rouge = PAS de signal optique (problème !) LOS → ⚫ Éteint = Signal optique OK LAN → 🟢 Clignotant (trafic) ou fixe (lien actif) ✓ PON vert fixe = ONT authentifié par OLT Orange ✗ LOS rouge = Pas de lumière → vérifier PTO et jonctions 4. Connecter câble RJ45 ONT LAN → port WAN Livebox/routeur 5. Vérifier attribution DHCP WAN → Internet fonctionnel
07

Test de débit et validation finale

Effectuer un test de débit complet pour valider l'offre souscrite et finaliser la remise en main client.
