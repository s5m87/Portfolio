Comprendre le FTTE

FTTE — Fibre mutualisée pour entreprises en immeuble

Le FTTE (Fiber To The Enterprise) désigne le déploiement de fibre optique jusqu'à chaque étage d'un immeuble tertiaire (bureaux, commerces), avec une distribution vers chaque locataire via un réseau interne structuré. C'est une solution intermédiaire entre le FTTH résidentiel et le FTTO dédié.

La fibre arrive en pied d'immeuble depuis le NRO Orange, puis est distribuée vers chaque étage à travers les gaines techniques verticales. Un switch de distribution en étage assure la connectivité des postes et équipements téléphoniques. Chaque locataire est isolé via un VLAN dédié.

  • Une seule infrastructure fibre partagée entre tous les locataires
  • Isolation complète par VLAN 802.1Q entre entreprises
  • Switch PoE+ pour téléphones IP et bornes WiFi sans câble d'alimentation
  • Débits garantis par QoS inter-VLAN
  • Gestion centralisée via portail Orange Business

Notice d'installation FTTE
01

Analyse du câblage et plan d'immeuble

Récupérer le plan de câblage de l'immeuble, identifier les gaines techniques disponibles et valider le parcours des câbles fibre avant toute intervention.

Points à vérifier avant démarrage
✓ Plan de câblage immeuble (gaines, fourreaux, colonnes) ✓ Emplacement du local technique (DTIo / NRO immeuble) ✓ Identification des étages et locataires à desservir ✓ Capacité des gaines techniques (place disponible) ✓ Vérifier présence baie de brassage en pied d'immeuble ✓ Accès aux locaux techniques (badge, clé syndic)
02

Raccordement dans le répartiteur immeuble (DTIo)

Connecter le câble de transport Orange dans le DTIo (Distribution de Télécommunication d'Immeuble optique) et réaliser le brassage vers les fibres de colonne montante.

Raccordement DTIo
✓ Localiser le câble de transport Orange (gaine extérieure) ✓ Dénuder et préparer les fibres (protection, nettoyer) ✓ Épissurer ou connecter sur la cassette ODF du DTIo ✓ Jarretiérer vers les fibres de colonne montante (1 fibre/étage) ✓ Étiqueter chaque jarretière : ETAGE-02-VL10, etc. ✓ Mesure OTDR depuis DTIo vers chaque étage Atténuation cible : < 3 dB par étage (câble G.657 OS2)
03

Tirage de la fibre dans les gaines techniques

Tirer les câbles fibre depuis le DTIo en pied d'immeuble jusqu'à chaque étage en passant dans les gaines techniques verticales.

Tirage fibre colonne montante
CÂBLE : OS2 (G.652D) — fibre monomode 9/125 µm ✓ Utiliser un câble multifibres (4 à 12 fibres) pour colonne ✓ Introduire câble avec aiguille de tirage / pistolet air ✓ Tension de tirage max : 1000 N (câble armé) ✓ Protéger chaque percement de gaine avec fourreau PVC ✓ Respecter rayon de courbure min : 20 × diamètre câble ✓ Fixer tous les 2 mètres avec colliers inox ✓ Laisser 1 mètre de mou à chaque étage (lovage) ✓ Étiqueter chaque étage : câble + numéro fibre REPÉRAGE COULEUR GAINE DIN : 🟦 Bleu → Fibre 1 (étage 1) 🟧 Orange → Fibre 2 (étage 2) 🟩 Vert → Fibre 3 (étage 3) [etc.]
04

Installation du switch PoE+ en étage

Installer le switch de distribution dans l'armoire de brassage de l'étage, câbler les ports SFP uplink en fibre et les ports cuivre RJ45 vers les postes de travail.

Installation switch étage — Cisco Catalyst C9200-48P
✓ Monter switch dans armoire 19" de l'étage (ou murale) ✓ Brancher alimentation secteur (230V / redondée si dispo) ✓ Connecter port SFP Uplink1 → fibre colonne montante (OS2) ✓ Connecter port SFP Uplink2 → redondance si possible ✓ Câbler ports RJ45 → patch panel → prises RJ45 bureaux PoE BUDGET ÉTAGE : Budget PoE total Cisco C9200-48P : 740W Téléphone IP Cisco : 8W par port Borne WiFi Cisco AP : 25W par port → Max ~25 téléphones + 4 bornes simultanément CÂBLAGE CUIVRE POSTES : ✓ Cat. 6A minimum (performance Gbit) ✓ Longueur max patch + horizontal : 100 m ✓ Test avec certifieur Fluke (NEXT, atténuation)

06

Tests de continuité et validation

Valider le câblage cuivre avec un certifieur, les liaisons optiques avec l'OTDR, et les VLANs par des tests de ping inter-VLAN.

Validation complète
TEST OPTIQUE UPLINK : OTDR : < 3 dB par étage sur colonne Powermètre uplink : dans seuil ONT/switch TEST CUIVRE POSTES (Fluke DTX-1800) : NEXT : > 43 dB (Cat. 6) Atténuation : < 21 dB à 250 MHz Return Loss : > 20 dB TEST VLAN (depuis PC sur VLAN 10) : ping 192.168.10.1 → OK (passerelle) ping 8.8.8.8 → OK (internet) ping 10.0.20.5 → ÉCHEC (inter-VLAN bloqué si voulu) TEST PoE TÉLÉPHONE : Brancher téléphone IP → boot PoE automatique show power inline Gi1/0/25 → On / 7.8W RÉSULTAT → Saisir dans CR GLPI + photos
Dépannage fréquent FTTE
Uplink SFP ne monte pas
Module SFP incompatible (non Cisco), ou fibre OS2 mauvaise longueur d'onde (1310nm vs 1550nm)
→ Vérifier compatibilité SFP (Cisco certified) → Vérifier type fibre (OS2 pour SFP LX/LH)
Téléphone ne démarre pas (PoE)
Budget PoE du switch dépassé, ou port non configuré en "power inline auto"
→ show power inline → Vérifier budget total → Réduire devices PoE ou ajouter switch PoE
PC sur mauvais VLAN
Port en mode trunk au lieu de access, ou VLAN ID incorrect sur le port access
→ show interfaces Gi1/0/X trunk → Vérifier switchport access vlan 10
Spanning tree bloque un port
Boucle physique détectée, ou portfast non activé sur port d'accès
→ show spanning-tree → Activer spanning-tree portfast sur tous les ports access
