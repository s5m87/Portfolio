## 📂 Structure du portfolio

### Pages principales

| Page                | Contenu                                                                          |
|---------------------|----------------------------------------------------------------------------------|
| `index.html`        | Page d'accueil — présentation rapide, photo et navigation                        |
| `about.html`        | À propos & Formation — parcours, certifications, objectifs                       |
| `competences.html`  | **Bilan par compétence** (RT1, RT2, RT3, Cyber) — référentiel BUT R&T            |
| `stack.html`        | Stack technique — technologies et outils maîtrisés (barres de progression)       |
| `orange.html`       | Orange Business — présentation entreprise, 4 missions cliquables, projection pro |
| `projets.html`      | Projets académiques (SAÉ, TP, projets individuels)                               |
| `contact.html`      | Coordonnées et formulaire de contact                                             |

### Pages mission détaillées (dans `/pages/`)

| Page                          | Mission                                                            |
|-------------------------------|--------------------------------------------------------------------|
| `pages/mission-connect-pro.html` | **Mission 1** : Installation Connect Pro                         |
| `pages/mission-routeur.html`     | **Mission 2** : Installation de Routeur (Cisco/Huawei)           |
| `pages/mission-fibre.html`       | **Missions 3 & 4** : Prestations FTTH/FTTE + FTTO (regroupées)   |
| `pages/mission-ipbx.html`        | **Mission 5** : Installation IPBX (téléphonie d'entreprise)      |

Chaque page mission contient : architecture (diagramme SVG), description technique, compétences &
apprentissages critiques BUT mobilisés, analyse réflexive & autoévaluation, galerie photos terrain
et fiche technique d'installation rétractable.
